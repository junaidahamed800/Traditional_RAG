# Traditional_RAG
This Repository contians Traditional RAG pipeline workflow

* Any type of data format (Structured / UnStructured) like PDF'S, Word, Excel etc is Converted into Document Structure with the help of LangChain Document Loaders
- PDFLoader
- CSVLoader
- WebBaseLoader
- DirectoryLoader & etc..
* Document is one type of data structre which has Two core components - page_content (String format) & metadata(Dictionary format)
